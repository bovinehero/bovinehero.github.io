# Bovine Hero

Static personal site for Me - Cyber Risk Engineering, secure systems work, labs, and side projects.

This repository contains:

- `index.html` — main site template
- `css/main.css` — styles (light + dark themes)
- `js/main.js` — theme toggle + small helpers

## Development

No build step required.  
Just open `index.html` in a browser.

## File Structure

``` sh
/
├── index.html
├── css/
│   └── main.css
└── js/
    └── main.js
```

## Hosting

Designed for GitHub Pages.  
Ensure Pages is enabled in repository settings and pointed at the root branch.

## Custom Domain

If using `bovinehero.com`, add a `CNAME` file with:

```
bovinehero.com
```

Configure DNS to GitHub Pages A records.
